## 💡 Best Practices for Genuine Numbers

### ✅ DO THIS
- Ask "Why do you want to join?" - filters fake signups
- Require name + country - shows commitment
- Check if they've visited your website
- Look for LinkedIn/social profiles
- Start with friends/colleagues (seed members)
- Ask members to invite 1-2 friends
- Provide value BEFORE asking for number

### ❌ DON'T DO THIS
- Buy phone number lists (fake/illegal)
- Add people without permission
- Spam random groups
- Send too many messages
- Share member numbers
- Ignore privacy laws

## 📞 Phone Number Format

### Always Collect in International Format
```
Correct: +1 234 567 8900 (USA)
Correct: +91 98765 43210 (India)
Correct: +44 20 1234 5678 (UK)
Correct: +86 138 0000 0000 (China)

Wrong: 234-567-8900 (missing country code)
Wrong: 9876543210 (missing +91)
```

### Country Codes Reference
- USA/Canada: +1
- UK: +44
- India: +91
- Germany: +49
- France: +33
- Australia: +61
- Singapore: +65
- China: +86
- Japan: +81
- Brazil: +55
